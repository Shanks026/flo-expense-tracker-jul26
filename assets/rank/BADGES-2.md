# Rank Badges — Batch 2 of 3

Match the style of `assets/comeback/comeback.png` — hexagonal medal frame,
laurel wreath on both sides, a ribbon + star base, rich gradient background,
one illustrated animal as the centerpiece. Same frame family every time;
the animal and color change per rank.

This batch covers the middle 3 ranks — sharper, cleverer animals, cooler
and richer colors than Batch 1, still short of Batch 3's top-tier grandeur.

| # | id (filename) | Title (display) | XP | Animal | Color |
|---|---|---|---|---|---|
| 4 | `strategist` | Navigator | 4,000 | Hawk — sees the path from above | Olive green / sky blue |
| 5 | `treasurer` | Strategist | 10,000 | Arctic Fox — thinks a few steps ahead | Ice blue / white |
| 6 | `financier` | Architect | 22,000 | Antelope — precise, sharp, always graceful | Violet / amethyst |

## Files

Drop finished PNGs into `assets/rank/`. **Name files by id, not title** —
rank 5 is titled "Strategist" but its id is `treasurer` (don't confuse this
with rank 4's id, `strategist`, which is titled "Navigator") — use the `id`
column above, not the `Title` column, when saving:

`strategist.png`, `treasurer.png`, `financier.png`.

Each also needs a grayscale/locked twin for when the rank isn't yet reached
— same name + `-locked` (e.g. `treasurer-locked.png`).
