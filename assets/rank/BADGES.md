# Rank Badges

Match the style of the existing badge art already in this app (medal shape,
ribbon, illustrated icon) — same family, your creative call on the rest.

## Ranks — 9 tiers (a ladder)

A progression from a beginner saver to a financial master, based on
lifetime XP earned in the app. One badge per rank, escalating in richness
as the ladder climbs:

- Saver — 0 XP
- Bookkeeper — 1,500 XP
- Steward — 5,000 XP
- Strategist — 12,000 XP
- Treasurer — 25,000 XP
- Financier — 50,000 XP
- Tycoon — 90,000 XP
- Magnate — 150,000 XP
- Money Master — 250,000 XP (top tier)

## Files

Drop finished PNGs into this folder (`assets/rank/`), named after the rank:
`saver.png`, `bookkeeper.png`, `steward.png`, `strategist.png`,
`treasurer.png`, `financier.png`, `tycoon.png`, `magnate.png`,
`money_master.png`.
