# Rank Badges

Split into 3 batches of 3 ranks each, so they can be handed to an image
generator separately:

- [`BADGES-1.md`](./BADGES-1.md) — Saver, Keeper, Steward
- [`BADGES-2.md`](./BADGES-2.md) — Navigator, Strategist, Architect
- [`BADGES-3.md`](./BADGES-3.md) — Vanguard, Master, Sovereign

Each batch file is self-contained (style reference, its own 3-row table,
file-naming instructions) — no need to read this file or the others to use
one.
